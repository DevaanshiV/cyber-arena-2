# CyberArena

CyberArena is a gamified cybersecurity learning platform designed to help users develop practical cybersecurity skills through hands-on Capture The Flag (CTF) challenges.

## Features

* User Sign Up and Sign In
* Interactive CTF Challenges
* XP-Based Progression System
* Leaderboards
* Achievement Certificates
* Web Security Challenges
* Cryptography Challenges
* OSINT Challenges
* Digital Forensics Challenges
* Responsive User Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage

## Challenge Categories

### Web Security

Learn to identify common web application vulnerabilities and hidden information.

### Cryptography

Practice decoding and basic cryptographic analysis.

### OSINT

Discover information through publicly accessible resources.

### Digital Forensics

Analyze evidence files and investigate digital artifacts.

## Project Goal

CyberArena was developed to make cybersecurity education more engaging through gamification, practical challenges, and achievement-based learning.
